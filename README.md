# FundamentosIA
Proyecto final del módulo.
Esta aplicación permite detectar enfermedades en hojas de rosa usando el modelo YOLO y Streamlit.

## Características
- Permite la carga de imágenes o video.
- Permite la descarga de imágenes o video con las enfermedades detectadas.
